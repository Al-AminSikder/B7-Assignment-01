# Advanced Problem Solving with TypeScript & OOP

## Project Overview

This repository contains solutions for the "Advanced Problem Solving with TypeScript & OOP" assignment. The project demonstrates fundamental TypeScript concepts including:

- Functions
- Array manipulation
- Union types
- Type guards
- Generics
- Interfaces
- Object-Oriented Programming (OOP)
- Inheritance
- Type safety

The assignment also includes two technical blog posts written in Markdown format.


# Project Structure

```bash
├── solutions.ts
├── blog-1.md
├── blog-2.md
└── README.md
```

# Problem List

## Problem 1
Filter even numbers from an array using TypeScript functions.

## Problem 2
Reverse a string using TypeScript.

## Problem 3
Use union types and type guards to identify string or number values.

## Problem 4
Create a generic function to safely access object properties.

## Problem 5
Use interfaces and object manipulation to manage book information.

## Problem 6
Implement Object-Oriented Programming concepts using classes and inheritance.

## Problem 7
Find the intersection between two arrays.
